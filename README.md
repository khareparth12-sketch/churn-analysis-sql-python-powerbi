# Customer Churn Analysis – Telecommunications Industry

## Overview

This project analyzes customer churn patterns in a telecom company using exploratory data analysis, data cleaning, and segmentation techniques. The goal is to identify high-risk customer profiles and propose actionable retention strategies.

## Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- SQL

## Methodology

1. Data collection and initial exploration
2. Data cleaning and preprocessing
3. Feature analysis and segmentation
4. Exploratory data analysis
5. Insight generation and business recommendations

## Key Insights

- Nearly 48% churn observed among first-year customers.
- Month-to-month contracts showed the highest attrition.
- Electronic check users exhibited elevated churn.
- Mid-to-high pricing tiers faced increased customer dissatisfaction.
- Long-term contracts demonstrated strong retention.

## Business Impact

The insights from this project enable targeted retention strategies, improved customer lifetime value, and reduced acquisition costs through proactive churn management.
